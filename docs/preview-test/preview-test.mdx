## Previews Processing Team Requirements

The previews processing team would require the mentioned below to test out samples:

- **Brand Name**
- **Website Link**
- **Inputs of minimum**: 1500 x 1500 px, 72 dpi (Preferred input format: PNG, JPG, JPEG)
- **Paring Product Input** along with the primary product input
- **Information on End Use Case**: Internal or website
- **Sample Naming Convention** (optional)
- **Styling Instruction** (optional)
- **Output Dimension** (optional)
- **Bgcolor Change** (optional)
- **Model Brief or Pre-shot Client Models** according to brand preference (optional)
- **Packshot** (optional)

### VueModel does not support the following categories:

- Flat garment inputs
- Extreme editorial shots
- Avant-garde garments
- Shoes
- Cap
- Draped & extremely fluid garments
- Foreshortened garment input

### Please try to avoid the following for testing:

- TIFF and HEIC files
- Model images shot with hair down & any interaction with the product
- Inputs with harsh lighting
