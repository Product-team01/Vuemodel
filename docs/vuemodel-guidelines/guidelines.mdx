# Basic Product & Model Shooting Guidelines

## Basic Image Requirement

- All images provided by the client need to be of high quality for quality generation.
- Please avoid submitting blurred images without textures, images with poor focus, and overly compressed JPEG files.
- Images can be provided in ghost formats, transparent PNG, input on body-form. Flat lay and hanger should be avoided.
- For consistent output, the images provided should have a similar color temperature and tonality.
- Image formats such as JPEG, PNG, JPG, TIF are accepted file formats.
- We generate images in these standard resolutions: 960 x 1280, 1200 x 1600, 1920 x 2560, and 2304 x 3072. To achieve optimal results, please provide input images with similar resolutions.

## Garment Inputs on Mannequin

- Preferred Mannequin Color are White, nude shades 401 etc.
- Use of different Mannequin sizes for size visualization.
- Condition of garment should be clean and not crashed or Packard.
- Please do avoid paired garments on same body form.
- Full sleeves can be put on full hand mannequin, and side 90 can have a pose with hand apart.

# Model Shooting and Posing

- Model needs to be posing such that the model shoulder and foot aligning on a same line to the preferred angle.
- Best quality generation can be driven on simpler fluid poses then complex action poses.
- Model undergarments should be light in thickness for better generations (avoid loose fit boxers, tank tops).
- Pre-paired model wearing bottoms or top can also be provided by the client.
- Models will need to have the MUA arranged with desired hairstyle, shoe footwear options, accessories etc.

## Staging the Input Asserts

- Garment and model can be staged on compass-like marker kept on the floor for consistent output quality. (Notes: Using marker, tapes to mark the position on the floor).
- Vuermodel - Standard Angles are straight 0°, Side 45°, Side 90°.
- If client have specific angle to their guideline, making sure that both model and input have the same angles gives the most accurate output.
- Synchronize lighting of the inputs: Ensure that any input provided match the model's positions.
